POEM: Prediction Of immunogenic Epitopes using Mechanistic modelling

Machine learning is only as good as its training data. Existing predictive models of class I antigen presentation rely on semi-quantitative data from mass spectrometry, which is not optimal. We predicted that we could improve antigen presentation predictions by developing a mechanistic model of the underlying biochemistry.

The resulting model appears to enhance predictions. Furthermore, immunogenicity classifiers trained using these mechanistic predictions appear to outperform the state-of-the-art across a wide range of tasks.

Inference of MHC-I from peptide ligands

Immune cells present peptides on their cell surface in proteins known as MHC-I. The binding affinity and stability between peptides and MHC-I is key to the efficacy of the associated immune response, so predicting these quantaties is desirable. Predictive models are trained using peptide ligands eluted from the surface of cells and identified by mass spectrometry. However, humans have up to 6 different types (alleles) of MHC-I, so it is non-trivial to identify which peptides were bound to each MHC-I allele.

To address this, I fine-tuned a pre-trained protein language model (ESM-2) to predict MHC-I given the peptide sequence. The resulting model could accurately deconvolute multi-allelic mass spectrometry datasets.
